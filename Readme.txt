#  Prueba Tecnica Creative Jungle

##Descripción

Este repositorio contiene una prueba técnica de una aplicación desarrollada en React Native. La aplicación se ha creado como parte de un proceso de evaluación y demuestra mis habilidades en el desarrollo de aplicaciones móviles utilizando la tecnología React Native.

La prueba técnica consiste en una aplicación móvil desarrollada en React Native que cumple con los siguientes requisitos:

Diseño: Diseñado segun las medidas dadas por la empresa, siguiendo las especificaciones proporcionadas.
Eliminar Cards: función de eliminar para las cards.
Obtener los datos para los títulos y descripciones desde la API de JSONPlaceholder: https://jsonplaceholder.typicode.com/posts.
Integrar la API para cargar los datos de la tabla uno a uno.

La aplicación se ha diseñado y desarrollado siguiendo las mejores prácticas de desarrollo en React Native con Expo y haciendo uso de las herramientas y librerías pertinentes.


## Librerias Usadas


- Node.js > 16+
- Expo CLI 
- heroicons
- axios
- nativewind


## Instalación

1. Clona este repositorio: `git clone https://github.com/deividcool/prueba-tecnica-creative-jungle`
2. Navega al directorio del proyecto: `prueba-tecnica-creative-jungle`
3. Instala las dependencias: `npm install` o `yarn install`

## Uso

1. Ejecuta la aplicación con Expo: `npx start` o `expo start -c`
2. Escanea el código QR con la aplicación Expo Go en tu dispositivo móvil
3. La aplicación se cargará en tu dispositivo
